# 📘 产品设计说明书：Mediterranean Relay (地中海中继站)

**版本：** v1.0 | **角色：** 首席产品经理 & 视觉总监

---

## 一、 产品定位与愿景

- **核心定位：** 全球绝版二手黑胶专辑的“数码拍卖行/中继站”。

- **设计哲学：** **“有温度的机械主义”**。将 70 年代地中海复古美学与未来主义交互融合，为高端买家提供极具仪式感的交易体验。

- **目标人群：** 顶级黑胶收藏家、发烧友、追求“触觉数字化”的高净值用户。

---

## 二、 核心视觉与交互规范 (Design DNA)

### 1. 材质与色谱 (Palette)

| **元素**   | **色值/定义**                | **心理暗示**     |
| -------- | ------------------------ | ------------ |
| **基础背景** | `#E8E4D9` (象牙色)          | 陶瓷、石灰石、温暖的阳光 |
| **主交互色** | `#1A4B9E` (深海蓝)          | 稳重、专业、地中海深处  |
| **金属装饰** | `#C7C2B5` (风化银)          | 二手物品的时间沉淀感   |
| **高亮反馈** | `RGBA(255,255,255,0.85)` | 物理光影的直射感     |

### 2. 核心交互逻辑 (UX Principles)

- **软浮雕 (Neumorphism)**：界面元素遵循物理重力与光影逻辑，所有按钮必须有“按下”和“浮起”的物理反馈。

- **非线性转场**：页面切换应模拟机械面板的滑动或相机的快门开合，避免僵硬的左右滑动。

- **阻尼感 (Damping)**：滚动列表和拨盘控制需带有物理惯性和轻微的触觉反馈（Haptic Feedback）。

---

## 三、 功能模块设计

### 1. 首页：档案库 (Archive Collection)

- **智能滑块 (The Relay Slider)**：如 HTML 原型所示，顶部滑块用于快速筛选“年代”或“稀缺度”。

- **动态卡片**：
  
  - 买家照片自动加载“地中海滤镜”（暖色调+胶片感）。
  
  - 点击卡片：卡片伴随机械声效（可选）放大，背景轻微模糊。

### 2. 详情页：机械展台 (The Mechanical Stand)

- **3D 预览**：支持专辑封面的 3D 翻转，模拟买家亲手触摸实体唱片。

- **音频试听**：播放器采用虚拟“真空管”电平表。

- **磨损报告**：以机械图纸的形式展示专辑的瑕疵点。

### 3. 上架流程：连接中继站 (Linking)

- **AI 辅助录入**：扫描唱片矩阵码，自动匹配数据库中的历史价格和版本信息。

- **仪式感上传**：上传照片后，系统显示“正在进行视觉校准”，将用户照片自动融入应用材质。

## 四、 给开发的交互实现指南 (Technical Handover)

| **模块**   | **实现建议**                                                | **关键参数**          |
| -------- | ------------------------------------------------------- | ----------------- |
| **布局适配** | 使用 `viewport-fit=cover`                                 | 适配所有全面屏、刘海屏       |
| **阴影体系** | **外凸**：`5px 5px 12px #00000012, -5px -5px 12px #FFFFFF` | 严禁使用过重的黑色阴影       |
| **性能优化** | `backdrop-filter: blur(20px)`                           | 导航栏的磨砂玻璃需保持 60fps |
| **手势控制** | 下拉刷新需设计成“拉动机械弹簧”的动画                                     | 增加 15ms 的短振动反馈    |

---

## 五、 App 完整页面版图 (Roadmap)

1. **Splash Screen (启动页)**：
   
   - 画面：阳光下的一台复古唱片机零件缓慢拼装。

2. **Browse (发现页)**：
   
   - 即 HTML 原型页，支持流式加载。

3. **Vault (个人收藏箱)**：
   
   - 采用“抽屉式”布局，用户已购专辑像书架一样排列。

4. **Trade (交易中心)**：
   
   - 模拟一份签名的纸质合同，点击即“盖章”成交。
